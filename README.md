# 🍕 Calculadora Desktop con Electron

Aplicación de escritorio desarrollada con Electron, Node.js, HTML, CSS y JavaScript.

El proyecto simula un sistema de acceso mediante login y posteriormente permite realizar operaciones matemáticas básicas desde una interfaz amigable.

---

# 🚀 Tecnologías utilizadas

* Electron
* Node.js
* HTML5
* CSS3
* JavaScript
* Git
* GitHub

---

# ⚙️ Instalación y ejecución

## 1. Clonar repositorio

```bash
git clone https://github.com/Bet0os/calculadora-desktop.git
```

---

## 2. Entrar a la carpeta

```bash
cd calculadora-desktop
```

---

## 3. Instalar dependencias

```bash
npm install
```

---

## 4. Ejecutar aplicación

```bash
npm start
```

---

# 🖥️ Funcionalidades

## Ventana Login

* Captura de nombre y apellido
* Validación de campos vacíos
* Mensaje de bienvenida
* Acceso a la calculadora

---

## Ventana Calculadora

* Saludo personalizado
* Ingreso de números
* Operaciones matemáticas:

  * Suma
  * Resta
  * Multiplicación
  * División
* Visualización del resultado
